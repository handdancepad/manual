Guía de actualización de software pad

Compatibilidad PIU Rise

Descarga los archivos necesarios para actualizar tu pad, y sigue los pasos.

¡Importante! durante la instalacion del ide o la instalacion del codigo si te aparece un mensaje de que algun puerto com o dispositivo arduino leonardo intentando acreditar la confianza de windows tienes que aceptar

Descarga de archivos

1. Instala Arduino IDE

Descarga e instala el programa arduino-ide.exe desde el enlace de arriba antes de continuar.

2. Abre el archivo pad.ino con Arduino IDE

Haz doble clic o desde el menú de Arduino IDE, selecciona File → Open y elige pad.ino.

3. Conecta el PAD

Usa el cable USB y espera que el PAD sea reconocido

4. Selecciona el puerto correcto

En Arduino IDE, ve a Tools > Port y selecciona “(Arduino Leonardo)”.

5. Selecciona el board

Ve a Tools > Board > Arduino AVR Boards y elige Arduino Leonardo.

6. Verifica el código

Haz clic en el botón de “Sketch” luego en Verify/Compile para compilar el código y revisar que no haya errores.

7. Sube el programa al Arduino

Haz clic en el botón de “Sketch” luego en Upload y espera a que el proceso termine.

8. ¡Pad actualizado!

Ya puedes probar tu pad actualizado
